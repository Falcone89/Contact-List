# Contact List (Next.js 16)

Contact Manager app with SQLite.

### Quick Start
1. **Clone:** `git clone https://github.com/falcone89/contact-list.git`
2. **Install:** `npm install --legacy-peer-deps`
3. **Run:** `npm run dev`

### Notes
* **Database:** The `contacts.db` file is automatically created in the root directory upon the first run.
